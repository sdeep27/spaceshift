## Five modes

<CardGrid>
  <Card title="Prompt Manipulate" icon="random">
    Transform a prompt through 22 built-in operations — abstraction up/down, inversion, reflection, rotation, dimension shifts, translations — then optionally generate responses for each variant.
  </Card>
  <Card title="Prompt Tree" icon="list-format">
    Explore a prompt in three directions: **sub** (decomposition), **super** (abstraction), **side** (lateral). Configure depth per direction and render the full tree.
  </Card>
  <Card title="Prompt Chain" icon="rocket">
    Build a multi-turn conversation by adding followups one at a time. Each turn runs with full prior context.
  </Card>
  <Card title="Compare Models" icon="approve-check-circle">
    Run the same prompt across multiple models and rank responses with pairwise evaluation, auto-generated or custom metrics.
  </Card>
  <Card title="Grid Search" icon="setting">
    Sweep across models × transforms simultaneously and rank every combination to find what works best.
  </Card>
</CardGrid>

## Advanced: Python library

While the CLI is the primary interface, the underlying modules are also importable:

```python
from spaceshift import research_tree

result = research_tree(
    "How do general-purpose technologies reshape economic structures?",
    sub_n=[5, 3],      # 5 subtopics, each split into 3 more
    super_n=[5],       # 5 broader framings
    side_n=[3],        # 3 lateral perspectives
    search=True,       # web search for current information
)
```

This is unsupported — internal APIs may change without notice.
