A 5-minute tour from basic LLM calls to prompt exploration and evaluation.

## Basic LLM Usage

```python
from spaceshift import LLM

LLM().user("Explain quantum computing in one paragraph").chat()
```

## LLM as a Function

```python
sentiment = LLM().sys("Classify sentiment. Return JSON with key 'sentiment'.").user("{text}")
sentiment.run("I love this product!")    # {"sentiment": "positive"}
```

## Prompt Tree

Explore a topic in all directions — sub, super, and side:

```python
from spaceshift import prompt_tree

tree = prompt_tree(
    "What are the second-order effects of AI on labor markets?",
    sub_n=[5],
    super_n=[3],
    side_n=[3],
    viz=True,
)
```

## Prompt Transform

Apply transforms to explore prompt variations:

```python
from spaceshift import prompt_transform

result = prompt_transform(
    "What are the second-order effects of AI on labor markets?",
    transform="inverse",
)
```
