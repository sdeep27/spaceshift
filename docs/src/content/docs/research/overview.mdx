The core loop: **explore** prompt space in all directions, **manipulate** through transforms, **evaluate** to find what works best.

## Outline

- The three-phase loop: explore → manipulate → evaluate
- Explore: decompose topics with `subprompt`, `superprompt`, `sideprompt`, `prompt_tree`
- Manipulate: move through prompt space with `prompt_transform`, `language_transform`, `prompt_probe`
- Evaluate: rank outputs with `pairwise_evaluate`, `compare_models`, `grid_search`
- How the pieces compose together
- When to use which tool
- Structured markdown output as the common thread
